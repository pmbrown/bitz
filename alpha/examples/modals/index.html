Modals

Basic common styles for native HTML dialog and popover modal elements.

Examples (anchor)

The dialog examples use the Invoker Commands API so require manual closing with a button, the popovers use the Popover API so can be closed with a button or by clicking outside the popover*. If required either API can be used with both components. The button styles are not provided, the examples inherit those included with the forms & buttons.

Dialog

The quick brown fox jumps over the lazy dog.

The quick brown fox jumps over the lazy dog.

Example HTML 
<button commandfor="dialog" command="show-modal">Dialog</button>
  <dialog id="dialog">
  <h3>Dialog</h3>
  <p>The quick brown fox jumps over the lazy dog.</p>
  <button commandfor="dialog" command="close">Close</button>
</dialog>

<button popovertarget="popover">Popover</button>
<div id="popover" popover>
  <p>The quick brown fox jumps over the lazy dog.</p>
  <button popovertarget="popover" popovertargetaction="hide">Close</button>
</div>

*The Popover API also includes the manual popover state attribute [popover="manual"] attribute to disable clicking outside it to close it, this enables showing multiple popovers simultaneously.

Light and dark (anchor)

The .light and .dark utilities included with the content styles use the color-scheme property and the default modal colors to apply fixed light or dark values with no extra styling.
