Accordion

Accordion container styles for HTML details elements.

Examples (anchor)

The .accordion utility is applied to a container that includes multiple <details> disclosure elements to style them as a basic accordion component. If the name attribute is used with each disclosure as shown below the open element is closed when another is opened.
