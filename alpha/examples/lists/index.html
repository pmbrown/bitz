Lists

Style modifiers for HTML ordered, unordered and definition lists.

Examples (anchor)

Ordered and unordered lists (anchor)

Unstyled (anchor)

<ol class="ol-none">

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

<ul class="ul-none">

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Inline lists (anchor)

<ol class="ol-inline">

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

<ul class="ul-inline">

  • Item 1
  • Item 2
  • Item 3
  • Item 4

<ul class="ul-inline" style="--gap: 1.5rem;">

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Comma list (anchor)

<ul class="list-comma">

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Dividers (anchor)

<ul class="list-divider">

  • Home
  • About
  • Services
  • Contact

<ul class="list-divider" style="--divider: '/';">

  • Home
  • Section
  • Category
  • Page

<ul class="list-divider" style="--divider: '>';--divider-px: 0.25rem;">

  • Home
  • Section
  • Category
  • Page

<ul class="list-divider" style="--divider:'&bull;';--divider-px:0.75rem;">

  • Home

  • Services
  • Contact
