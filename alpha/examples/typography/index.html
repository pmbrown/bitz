Visually hidden (anchor)

The .vis-hidden and .vis-hidden-focus classes are utility classes for hiding textual content from the visual interface but keeping the text available for users with assistive browsing.

.vis-hidden, .vis-hidden-focus:not(:focus):not(:focus-within) {
  position: absolute;
  block-size: 1px;
  inline-size: 1px;
  margin: -1px;
  overflow: hidden;
  clip-path: inset(50%);
  white-space: nowrap;
}

Light and dark (anchor)

The .light and .dark are utility classes that use the color-scheme property to apply fixed light or dark colors to all the Fallback styles that include text, background or border colors property values.

.light {
  color-scheme: light;
}

.dark {
  color-scheme: dark;
}
